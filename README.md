# fluffy-train
a simple RESTful CRUD API using Spring-Boot, Hibernate and MySql.


## Explore Rest APIs

The app defines following CRUD APIs.

    GET /api/employee
    
    POST /api/employee
    
    GET /api/employee/{id}
    
    PUT /api/employee
    
    DELETE /api/employee/{id}

Test these requests using Postman or any other HTTP REST-client.
